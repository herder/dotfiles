# dotfiles

So... this is my dotfile repo. Might be useful to others with some slight tweaking, or might not be.

## Features, off the top of my head

- Everything vimified
- Homebrew for apps, via Brewfiles
- Uses `rbenv`, `pyenv`, `jenv`, and `nvm` to try to get some versioning sanity
- Uses Homesick to link dotfiles
- One repo for public settings, and one `secrets` Keybase repo for secrets
- Update everything in one go with `update_environment`
- Docker-machine
- Auto-install
